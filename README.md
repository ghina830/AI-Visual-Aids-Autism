# 🧩 AI-Assisted Visual Design for Autism Education

## 📝 Project Overview
This project leverages **Generative AI tools** and professional design platforms like **Canva** to create and customize visual educational aids. These tools are specifically designed for children with **Autism Spectrum Disorder (ASD)** to simplify complex concepts like "Interrogative Tools" through sensory-friendly and engaging visuals.

## 🤝 Clinical Collaboration
I collaborated closely with **Dr. Afraa Ahmad Dawood** (Speech and Language Pathologist) to ensure all visual assets met strict therapeutic and linguistic requirements for clinical use.

## 🛠️ Tech & Tools
* **Generative AI:** Used for generating precise visual concepts and high-quality assets.
* **Canva:** Utilized for professional layout design, formatting worksheets, and ensuring the tools are visually structured for educational environments.
* **Educational Design:** Focused on selecting appropriate color palettes and typography suitable for neurodivergent children.
